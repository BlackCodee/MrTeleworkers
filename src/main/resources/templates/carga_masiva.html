Usuarios sin devolver computador


¿Qué hacer para reportar un usuario que no ha devuelto su computador?

Si el usuario no ha entregado el computador en el tiempo estimado usted debe:

1. Ingresar los datos del usuario a un archivo CSV.
2. Los datos que se le solicitarán del usuario son: ID, NOMBRE, DIRECCION, CIUDAD, ESTADO, ROL, CELULAR Y CLAVE.
3. El archivo CSV debe subirse por el apartado de "Seleccionar Archivo".
